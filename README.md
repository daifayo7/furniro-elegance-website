## 📖 项目简介 | Overview

**中文**  
本项目是一个全静态、高性能的家具家居品牌展示官网，**100% 由 OpenCode 调用 Anthropic 官方 `frontend-design` Skill 生成**。从设计指令到组件落地均未手写一行基础结构代码，旨在验证 AI 辅助工程化在商业级前端项目中的落地效率与美学上限。

**English**  
A fully static, high-performance furniture brand showcase website, **100% generated by OpenCode using Anthropic's official `frontend-design` Skill**. No boilerplate HTML/CSS was manually written—this project validates the efficiency and aesthetic ceiling of AI‑assisted engineering in production‑grade frontend development.

## ✨ 核心亮点 | Key Features

### 🎨 1. Anthropic 官方设计技能驱动 | Driven by Anthropic's Official Design Skill  
**中文**  
通过 OpenCode 调用 Anthropic 发布的 [`frontend-design`](https://github.com/anthropics/skills/tree/main/skills/frontend-design) 技能（GitHub 117.8k+ Stars），AI 严格遵循**非通用型美学标准**生成界面，彻底摆脱千篇一律的“AI 感”布局。

**English**  
The UI is generated via OpenCode using Anthropic's official [`frontend-design`](https://github.com/anthropics/skills/tree/main/skills/frontend-design) Skill (117.8k+ GitHub Stars). The skill enforces **non‑generic aesthetics**, eliminating the bland, predictable style common in AI‑generated interfaces.

### ⚡ 2. 拒绝“AI 感”的精致视觉 | Refined Visuals, Zero "AI Slop"  
**中文**  
- 非对称网格与大量留白构建呼吸感。  
- 定制化悬停动效、贝塞尔曲线过渡以及原生表单重设计。  
- 移动端/平板/桌面三端独立响应式策略，非简单缩放。

**English**  
- Asymmetric grids and generous white space for breathing room.  
- Custom hover states, bespoke easing curves, and redesigned native form elements.  
- True responsive design with distinct layouts for mobile, tablet, and desktop.

### 🚀 3. 静态站点的性能天花板 | Performance Ceiling for Static Sites  
**中文**  
零 JS 框架运行时开销；图片资源遵循 `frontend-design` 建议进行 WebP 转换与懒加载。Lighthouse 四项评分均接近满分。

**English**  
Zero runtime overhead from JS frameworks. Assets are optimized (WebP + lazy loading) as recommended by the `frontend-design` skill. Lighthouse scores are near‑perfect across all four categories.

### 🔄 4. AI 原生的工程化部署闭环 | AI‑Native CI/CD  
**中文**  
OpenCode 生成 → Git 提交 → GitHub 仓库同步 → Vercel 自动构建部署。代码变更后 **10 秒内**即可刷新线上站点。

**English**  
OpenCode generation → Git commit → GitHub sync → Vercel auto‑deployment. Live site updates **within 10 seconds** of a code push.
